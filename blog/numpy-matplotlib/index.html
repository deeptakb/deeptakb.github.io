Plotting deformation of a two dimensional grid

Here we demonstrate that the density only changes if the divergence of the strain tensor \(u\) is non-zero. \[ \] \[ \nabla \cdot u \neq 0\] \[ \] This is essentially because of the fact that the change in volume is related to the divergence of the strain.

Here we show that if the deformation is non-trivial then the body can take interesting shapes. Also, note that if the divergence of \(u\) is zero then the body has only shear deformation but the density in the bulk remains constant. It is only in the case when the divergenc is non-zero we see the effect on this deformation on the bulk of the body.

Subplot in the 3d plots using matplotlib.

Here we give a demo of subplots in 3d using matplolib. The disturbances are are due to two different initial disturbances.
In [19]: 
%matplotlib inline
from mpl_toolkits.mplot3d.axes3d import Axes3D
from matplotlib import cm
import matplotlib.pyplot as plt
import numpy as np

xx = np.linspace(-4, 4, 80)
yy = xx
x, y = np.meshgrid(xx, yy)

f = plt.figure(num=None, figsize=(15, 10), dpi=80, facecolor='w', edgecolor='k')

def plot_3dG(c_, n_):
    ax = f.add_subplot(2, 2, n_, projection='3d', )
    r = np.sqrt(x**2 + y**2)
    c = c_
    r1 = abs(r + c)
    r2 = abs(r - c)
    heights = 0.5*(np.exp(-r1*r1) + np.exp(-r2*r2))
    ax.plot_surface(x, y, heights, rstride=1, cstride=1, cmap=cm.coolwarm,
            linewidth=0, antialiased=False)

c, n = 0, 1           #plot 1
plot_3dG(c, n)

c, n = 2, 2           #plot 2
plot_3dG(c, n)

import matplotlib.pyplot as plt

xx = np.linspace(-4, 4, 80)
yy = xx

    ax = f.add_subplot(2, 2, n_, projection='3d', )

    c = c_

            linewidth=0, antialiased=False)
